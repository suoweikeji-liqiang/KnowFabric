# Architecture Decision Records (ADR)

This directory contains architectural decisions made for KnowFabric.

## What is an ADR?

An Architecture Decision Record (ADR) captures an important architectural decision along with its context and consequences.

## Format

Each ADR follows this structure:

1. **Title** - Short noun phrase
2. **Status** - Proposed, Accepted, Deprecated, Superseded
3. **Context** - What forces are at play
4. **Decision** - What we decided to do
5. **Consequences** - What becomes easier or harder
