## 契约

- consensus_state 6 enum 终态: `agreed / partial_conflict / value_disagreement
  / over_merge / material_conflict / single_source / undetermined`
- value_disagreement + fallback_highest_rank 触发 server 主动 null 顶层 range
  + range_disclaimer flag (强制消费方走 authority_layers)
- redistribution_restricted=true 触发 citation-only stub (verbatim 屏蔽)
- min_trust_level 默认 L1 (最宽松，含全部)
- /fault-knowledge 是 union endpoint (返回 fault_code + fault_diagnostic_rule
  + diagnostic_step)
